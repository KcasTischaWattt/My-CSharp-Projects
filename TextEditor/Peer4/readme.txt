Некоторые замечания по поводу Notepad--:
1) Если файл существует - он сохраняется сам в себя. Если не существует - то открывается диалоговое окно
2) 1 пункт не относится к Save As
3) Если файл не сохранен - то у его имекни стоит звездочка. Это очень удобно для проверки автосохранения
4) Автосохранение работает только на существующие в файловой системе файлы
5) К файлам, не существующим в файловой системе(с дефолтным именем NewFile) звездочка не добавляется
6) Настройки приложения, а так же открытые файлы сохраняются в json только у последней открытой формы. 
То есть, если было открыто 2 формы, сначала была закрыта первая, а потом вторая, то восстановятся при перезапуске только файлы, открытые до этого во второй форме
7) В json записываются только существующие в файловой системе файлы
8) Горячие клавиши:
ctrl + s - сохранить текущий файл
ctrl + s + shift - сохранить все файлы
ctrl + o - открыть файл
ctrl + n - новая вкладка
ctrl + n + shift - новое окно
ctrl + w -закрыть окно
ctrl + w + shift - закрыть файл
ctrl + b - жирный шрифт
ctrl + i - курсив
ctrl + u - подчеркнутый шрифт
ctrl + z - undo
ctrl + z + shift - redo
9) Если вы откроете слишком много форм - то переполнится оперативная память. От такого падают все программы - это системная ошибка, я не знаю, как такое предотвратить
